A Tailwind-free CSS framework, powered by LLM assistants

The Old Skull ☠️ CSS framework combines handcrafted CSS architecture with AI-powered generation – so every component stays clean, consistent, and ready for production.

Let's Rock
Are you already “too old for this shit”? Tired of the “CSS-in-JS soup” and endless Tailwind utility chains? But still believe in semantics – and love adopting new CSS features as soon as Can I Use hits 90%? Well, buddy, welcome home – to our cozy, old-school-cool Bootstrap Old Skull. A CSS framework and boilerplate built together with ChatGPT, Claude, and Gemini. Packed with cutting-edge CSS features – and total creative freedom. You’re not stuck tweaking variables here – grab any component and restyle it to fit your design vision. It’s free!

Quick Start

Copy the entire repository and use it as a project starter.

git clone https://github.com/supervova/old-skull.git
  • Enable or disable the necessary files inside frontend/main.css.
  • Adjust values in frontend/theme/vars*.css.
  • Rewrite global styles and component styles freely – if you know CSS well, there’s no need to limit yourself to variables or simple overrides. CSS is one of the most creative areas in software development.
