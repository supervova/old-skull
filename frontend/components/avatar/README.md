# Avatar

An avatar is a UI component that displays a profile image, initials, or icon to visually represent a user, company, organization, or group.

Usage:

```html
<!-- Use `span` inside `summary…` -->
<span class="avatar" data-fallback="VN"></span>

<!-- …and figure in all other cases. -->
<figure class="avatar avatar-xl">
  <span class="tooltip" data-tooltip="Vladimir 'Old Skull' Nikishin, author">
    <img src="/assets/img/i-am-madison-av.jpg" alt="Vladimir Nikishin, author">
  </span>
</figure>
```

Используй `span` в `summary`…
…и `figure` в остальных случаях
